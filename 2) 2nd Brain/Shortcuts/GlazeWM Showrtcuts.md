# Navigation
Alt + {h j k l}
- Changes Focus to window in that direction
Alt + {1 2 3 4 ... 9}
- Changes Focus to that workspace
Alt + d
- Desktop
# Moving & Editing Windows
Alt + shft + {h j k l}
- Moves window that direction
Alt + {u i o p}
- Grows window that direction
Alt + shft + {1 2 3 4 ... 9}
- Moves window to that workspace
Alt + w
- Closes that window
