# Tabs
Ctrl + tab
- Cycle Between Recent Tabs
Ctrl + t
- Creates new tab
Ctrl + shft + t
- Reopens most recent closed tab
Ctrl + w
- deletes current tab
Ctrl + arrow keys
- changes to tab in that direction
# History
Ctrl + h
- Opens History
Alt + left (arrow key)
- Go back to previous
Alt + right (arrow key)
- Go forward to next (in history)
